# Проект: Mesto

## Описание

Учебный проект "Mesto".
Подготовка скелета для дальнейшей разработки. Выполнен на **HTML**, **CSS**, **JavaScript**

## Установка

Скачать, распаковать, при установленном **node.js** запустить **npm i** затем **npm run dev**
